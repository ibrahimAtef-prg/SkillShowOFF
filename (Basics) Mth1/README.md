# Math Curriculum — What You Need for AI/ML

## ✅ Essential — Study These

| # | Topic | Why It Matters |
|---|-------|---------------|
| 1 | Real Functions | Understanding inputs, outputs, loss functions |
| 2 | Limits & Continuity | Foundation for derivatives |
| 3 | Differentiation | This IS backpropagation — most critical topic |
| 5 | Integration | Probability distributions, expectations |
| 6 | Linear Algebra | Everything in ML runs on this |

---

## ⚠️ Partial — Know the Basics Only

| # | Topic | What to Take |
|---|-------|-------------|
| 9 | Series (conv. & div.) | Taylor series only — needed for understanding gradient approximations |
| 10 | Fourier (Series / Transform) | Basic intuition only — useful later in signal ML |

---

## ❌ Skip for Now — Not Needed in ML

| # | Topic | Reason |
|---|-------|--------|
| 4 | Partial Fraction & Long Division | Pure algebra — no ML application |
| 7 | Conic Sections | Geometry — no ML application |
| 8 | Differential Equations (ODE / HDE) | Only needed for very advanced research |
| 11 | Laplace Transform | Not used in standard ML |

## Your Order of Study

```
1. Real Functions
2. Limits & Continuity
3. Differentiation        ← spend the most time here
4. Linear Algebra         ← study this in parallel
5. Integration
6. Taylor Series (from topic 9)
```

## Done When

- [ ] Can define a function and identify its domain
- [ ] Can compute a limit by hand
- [ ] Can differentiate any standard function
- [ ] Can apply the chain rule (this = backprop)
- [ ] Can multiply matrices and compute dot products
- [ ] Can compute a basic integral
- [ ] Can explain what a Taylor series approximates
